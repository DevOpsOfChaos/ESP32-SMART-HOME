# Projektüberblick

## Ziel
Neuaufbau eines modularen Smart-Home-Systems für das Technikerprojekt.

## Leitidee
Nicht das alte Projekt weiterflicken, sondern eine saubere, veröffentlichbare und erweiterbare Basis schaffen.

## Systemebenen
- Nodes auf ESP32-Basis
- Master als ESP-NOW <-> MQTT Bridge
- Raspberry Pi Server für MQTT, Node-RED und spätere Visualisierung

## Feste Grundprinzipien
- vier Basisgeräte statt Gerätewildwuchs
- keine private Konfiguration im öffentlichen Repo
- Dokumentation und Code wachsen gemeinsam
- Altprojekt nur als Referenz
- neue Basis ist GitHub-tauglich
